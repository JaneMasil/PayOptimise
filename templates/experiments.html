Methodology — causal inference framework
Hash-based randomisation
MD5 hash of (experiment_name + session_id) mod 100. Deterministic — same user always gets same variant. 50/50 split enforced. No centralised server needed. (Imbens & Rubin, 2015)
Chi-square + Cohen's h
Two-sided chi-square test for binary outcomes (conversion). Effect size: h = 2 × (arcsin(√p_t) − arcsin(√p_c)). Threshold: α = 0.05. SRM detection: >2% split deviation flagged. (Quin et al., 2024)
Guardrail metrics
Secondary safety indicators monitored alongside primary conversion metric. A significant but harmful treatment is blocked from shipping. Critical in financial contexts where design decisions affect real money. (Quin et al., 2024)
